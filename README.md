# HamroUpachar
BCA 4th Semester Minor Group Project by Advanced College of Engineering and Management- Team Members: Milan Aryal and Diksha Ghimire. 

Instructions for viewing this project:
1. Download or Clone this project into xampp/htdocs folder
2.Go to localhost/phpmyadmin and create a database 'minorproject' and inside this databse go to Import and choose the 'minorproject.sql' file which is located in 'Hamroupachar/Database' Folder
3.Type localhost/hamroupachar/frontend in any browser to view the project, but make sure that Xampp is running in your computer
